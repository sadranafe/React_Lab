# Slider (React + Vite)
this is a simple slider app built in React and using Vite and tailwind Css technologies.

## Technologies Used
- `React` : A JavaScript library for building user interfaces.

- `Tailwind CSS` : A utility-first CSS framework for rapidly building custom designs.

- `Vite` : A fast build tool for modern web applications.

## Getting Started
To get started with the slider app, follow these steps:

1. Clone the repository: 

```
git clone https://github.com/sadranafe/React_Lab.git
```

2. Navigate to the project directory: 

```
cd 002/slider
```

3. Install the dependencies: 

```
npm install
```

4. Start the development server:
```
npm run dev
```
5. click on O to open your browser : http://localhost:3000 to see the app in action.
